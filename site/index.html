    Project Folder Structure/Contents

    The project folder scRNAseqNilisha has the following subfolders and relevant contents as noted below;

    1. Archive : Old/unwanted files are stored in this folder, for backup purpose
    2. index.html : The main page to all project documentation
    3. Protocol : Contains experiment/protocol details
    4. RawData : Contains all the raw data obtained from the sequencing core
    5. Reports : Contains all the reports, including this documentation
    6. Results : Contains all the results
    7. Tools : Contains all the scripts and reference files
